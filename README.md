## Запуск проекта

Клиент:
### `cd client`
### `npm run install`
### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

Сервер:
### `cd server`
### `npm run install`
### `npm start`

Запустится сервер по адресу [http://localhost:8888](http://localhost:8888).
